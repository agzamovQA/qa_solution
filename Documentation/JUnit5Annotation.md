## Аннотации в JUnit5

### **WIP**

- `@BeforeEach` - метод, который выполняется перед тестом (Предусловие).
- `@AfterEach` - метод, который выполняется после выполнение теста. 


- `@BeforeAll` - метод, который является пре-реквезитом к выполнению всех тестов. И вызывается 1 раз всего. А не в каждом тесте, как предыдущие 2 Each метода.
  Например, настроить браузер сразу для всех тестов.
- `@AfterAll` - метод, который выполняется после выполнения всех тестов. И вызывается 1 раз всего. А не в каждом тесте, как предыдущие 2  Each метода.
  Например, настроить браузер сразу для всех тестов.


- `@Test` - запускает тесты
- `@DisplayName` - Аннотация для теста (можно описать название теста)
- `@Disabled` - отключает тест из Пайпа (Можно оставить аннотацию на номер в джире)
- `@Tag` - позволяет разметить тесты
- `@Tags` - позволяет проставлять сразу несколько тэгов.
- `@Order` - параметризация тестов
- `@ValueSource` - определяет сколько тестов будет запущено. Там можно добавить разные тестовые данные и сколько тестовых данных дбавишь, то столько и запустятся тестов
- `@CsvSource` -- можно добавить тестовые данные сразу с ссылкой и в заголовке отразить
- `@EnumSource` -- позволяет проверять описание на различных языках
- `@MethodSource` -- множество проверок
