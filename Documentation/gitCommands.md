## Комманды GIT-а
- `git --version` - узнать версию
- `git config --global user.name` - узнать Глобальную конфигурацию пользователя
- `git config --global user.email` - узнать Глобальную конфигурацию имейла
- `git config --local user.name` - узнать Локальную конфигурацию пользователя
- `git config --local user.email` - узнать Логальную конфигурацию имейла
- `git status` - покажет в каком статусе находится сейчас репозиторий
- `git init` -  инициализация гита в репозитории
- `git branch -m <file>` - отвести ветку от Мейна с определенным наименованием
- `git branch -M <file>` - переименовать бранч
- `git checkout <name branch>` - сменить ветку
- `git add <file>` - добавить файл в отслеживания Гитом, чтобы потом его закоммитить
- `git rm -f --cached .<file>` - убрать файл из отслеживания Гитом
- `git commit -m <comment>` - сделать коммит с комментарием
- `git push -u origin <branch name>` - используется при первичному пуше в ветку, чтобы в гите она пролилась
- `git fetch` - скачать последние изменения из удаленного репозитория, но не удалять их
- `git pull` - скачать последние изменения из удаленного репозитория и применить их
- `git remote add origin git@github.com:agzamovQA/qa_solution.git` - Задать обращение гита к конкретному удаленному репозиторию
